# Privacy Policy for Tab Silence: New Tab Blocker

Last updated: June 10, 2026

## Overview

Tab Silence: New Tab Blocker is a Chrome extension that helps users block websites from opening new tabs. The extension works locally in the browser and does not send browsing data to any external server.

## What data the extension uses

The extension may access the current website URL or base URL so it can:

- determine which website rule should apply
- detect when a website tries to open a new tab
- store the user's on/off preference for that website
- store the lifetime `2nd-Layer Block` counter for that website

## What data the extension stores

The extension stores the following data in Chrome extension storage on the user's browser:

- per-website blocking preferences
- whether blocking was previously enabled for a website
- whether popup-opening behavior was detected for a website
- lifetime backup-block counts for a website

This data is stored locally on the user's device and is used only to provide the extension's core feature.

## What data the extension does not collect

Tab Silence: New Tab Blocker does not:

- sell user data
- transfer browsing data to third parties
- use analytics, ads, or tracking services
- read page content for advertising or profiling
- execute remote code

## Network behavior

The extension does not send user browsing data to a developer-controlled server.

When blocking is enabled for a website, the extension may use Chrome's built-in extension APIs to stop or close popup tabs and to reduce request-header leakage during blocked popup attempts.

## User control

Users control blocking on a per-website basis from the extension popup. Users can also turn the extension off or remove it at any time through Chrome's extension settings.
